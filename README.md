📹 Prototype Demonstration

This project has been successfully implemented as a functional AI prototype integrating object detection, risk prediction, and automated alert generation.

Due to hardware and live deployment constraints, the current submission demonstrates a fully integrated software-level pipeline rather than a real-time deployed camera system.

The implemented prototype verifies:

Real-time object detection workflow using YOLOv8

Accident severity risk scoring logic

Threshold-based emergency alert triggering

End-to-end modular integration

The system is deployment-ready and can be connected to live CCTV or dashcam feeds in a production environment.
