# 🚀 ASTRA — AI System for Threat Recognition & Avoidance

## 📌 Overview
ASTRA is an AI-powered crowd intelligence and disaster prevention system designed to detect and predict dangerous crowd situations in real time. The system transforms traditional surveillance into an intelligent safety solution that helps authorities prevent accidents before they occur.

## 🎯 Problem Statement
Crowd-related incidents such as stampedes and overcrowding occur due to delayed monitoring and lack of predictive analysis. Existing CCTV systems only record events but cannot identify risks early.

## 💡 Solution
ASTRA uses computer vision and artificial intelligence to analyze live video feeds, detect crowd density, and generate early risk alerts. The system provides real-time insights through an interactive dashboard to enable proactive decision-making.

## ⚙️ Features
- Real-time people detection using AI
- Crowd density analysis
- Risk level prediction (Safe / Warning / Critical)
- Live monitoring dashboard
- Early alert system

## 🧠 Technology Stack
- Python
- OpenCV
- YOLO Object Detection
- TensorFlow / PyTorch
- Streamlit Dashboard
- NumPy & Pandas

## 🖥️ Demo
The dashboard displays crowd count, congestion levels, and safety alerts based on AI analysis.

## 🚀 Future Scope
- Edge deployment on smart cameras
- Smart city integration
- Emergency response automation
- Predictive analytics using historical data
